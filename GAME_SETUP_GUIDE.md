# 🎮 撿金幣遊戲 - 設置指南

## 📋 已創建的文件

### 遊戲邏輯
- ✅ `scripts/player.gd` - 玩家控制（移動、跳躍）
- ✅ `scripts/coin.gd` - 金幣邏輯
- ✅ `scripts/enemy.gd` - 敵人邏輯（追蹤玩家）
- ✅ `scripts/game_manager.gd` - 遊戲管理和得分系統

### 場景文件
- ✅ `scenes/main_scene.tscn` - 主遊戲場景
- ✅ `scenes/coin.tscn` - 金幣場景
- ✅ `scenes/enemy.tscn` - 敵人場景

---

## 🚀 在 Godot 中使用

### 第一步：打開 Godot 引擎
1. 下載並安裝 [Godot 4.0+](https://godotengine.org/)
2. 打開 Godot
3. 選擇「打開項目」
4. 選擇 `godtocode` 資料夾

### 第二步：設置主場景
1. 打開 `scenes/main_scene.tscn`
2. 在場景編輯器中添加背景和金幣
3. 設置玩家和敵人的位置

### 第三步：測試遊戲
1. 點擊「播放」按鈕（或按 F5）
2. 用方向鍵或手把控制玩家
3. 收集金幣並躲避敵人

---

## 🎮 遊戲說明

### 控制方式
- **方向鍵** 或 **手把搖桿** - 左右上下移動
- **空白鍵** 或 **手把 A 按鈕** - 跳躍

### 遊戲規則
- 🪙 收集所有金幣得到最高分
- 👾 避免敵人（紅色方塊）
- 💀 被敵人抓到遊戲結束

---

## 🎨 自定義遊戲

### 1. 添加真實素材
將 `assets/images/` 中的圖片替換到場景中：
- `background.png` - 背景
- `player.png` - 玩家
- `coin.png` - 金幣
- `enemy.png` - 敵人

### 2. 調整遊戲參數
編輯 `scripts/player.gd`：
```gdscript
var speed = 300          # 玩家速度
var jump_force = -400    # 跳躍力度
var gravity = 800        # 重力
```

編輯 `scripts/enemy.gd`：
```gdscript
var speed = 150          # 敵人速度
```

### 3. 增加遊戲難度
- 增加敵人數量
- 增加敵人速度
- 減少金幣數量
- 增加地形障礙

## 🐛 常見問題

### Q: 遊戲不能運行？
A: 確保 `project.godot` 文件存在，並且場景文件路徑正確。

### Q: 敵人不追蹤玩家？
A: 確保玩家節點命名為 "Player"，敵人才能找到它。

### Q: 金幣不能收集？
A: 確保金幣節點有 Area2D 組件和碰撞形狀。
